# ENPIRE Website

Anonymous project website for the CoRL submission:

**ENPIRE: Agentic Robot Policy Self-Improvement in the Real World**

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run typecheck
npm run lint
npm run build
```

## Structure

- `src/app` - Next.js app route, page layout, and global styling.
- `src/components` - Interactive figures and policy/reset panels.
- `src/data` - Data used by article visualizations.
- `public` - Static images, videos, fonts, and generated reset artifacts.
- `scripts` - Data preparation and annotation utilities used by the website.
